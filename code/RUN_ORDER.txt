Internal sequence orchestrated by ovk_pipeline.py:

1. python run_publication_grade_ovk.py
   - Produces publication-grade outputs.
   - Also writes deprecated top-five-compatible and shock-robustness-compatible output surfaces using the upgraded estimator.
2. python make_final_appended_top5_pack.py
3. python ovk_nested_workflow.py
4. python run_sec_robustness.py --results-dir results --bootstrap-draws 1000 --clean
5. Optional: python create_full_math_appendix_pack.py
